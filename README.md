### Ubuntu

 - [libgc-dev](http://packages.ubuntu.com/search?keywords=libgc-dev&searchon=names)

### Mac OS X, Windows

 - [Boehm GC](http://www.hpl.hp.com/personal/Hans_Boehm/gc/)

* [cppcheck](http://cppcheck.sourceforge.net/)
